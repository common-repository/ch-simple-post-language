Easily set the HTML lang attribute based on post or page.

== Description ==

This plugin lets you easily set the HTML lang attribute for each post or page.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Simple Post Language settings screen to configure the plugin

== Screenshots ==

1. Set the HTML lang attribute for this post or page
2. Select languages to be displayed in the edit page dropdown selection
3. The HTML lang attribute output
